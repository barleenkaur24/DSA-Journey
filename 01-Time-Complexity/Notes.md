# Time Complexity

## Definition
Time complexity measures how the running time of an algorithm grows as the input size increases.

## Common Complexities
- O(1)
- O(log n)
- O(n)
- O(n log n)
- O(n²)
- O(2ⁿ)
- O(n!)

## Rules I 
- Ignore constants.
- Keep the highest-order term.
- Consecutive loops add.
- Nested loops multiply.
- Halving the input usually results in O(log n).
